作為一個可定制的主題，其提供了定制資源的能力，無論是定制 CSS 和 JS，還是引入第三方 CSS 和 JS。

<!--more-->

## 內部資源

你只需要創建和編輯 `assets/css/custom.css` 和 `assets/js/custom.js` 即可。

> 這些檔案會被打包成單個 CSS/JS 檔案，以減少 HTTP 的請求次數。

## 外部資源

任意的 CSS 和 JS 資源都可以通過 `customCSS` 和 `customJS` 引數進行引入。

```toml
customCSS = [
  "external-foo.css",
  "external-bar.css"
]

customJS = [
  "external-foo.js",
  "external-bar.js"
]
```
