## 前置条件

我們需要在 `content` 目錄下創建一個名為 `archives/_ index.md` 的檔案。

```toml
+++
title = "归档"
layout = "archive"
+++
```

## Site Parameters

| 名称 | 类型 | 默认值 | 描述
|---|:-:|:-:|---
| `archive` | Object | - | Archive.
| `archive.paginate` | Integer | `100` | Archive pagination.
| `archive.dateFormat` | Integer | `Jan 2` | Archive date format.
