# shell

## helloworld

编辑内容
```
#!/bin/bash
echo "hello world!"
```
保存退出:
`w ~/helloworld.sh`
运行:
```
chmod +x ~/helloworld.sh
cd ~
./helloworld.sh
```
执行结果:
`hello world!`

### 分析:

第一行中`#!`是一个约定的标记,告诉系统脚本需要使用什么解释器来执行,即使用哪一种shell

这种在第一行指定了解释器信息的方式,需要让脚本作为可执行程序执行

还有第二种运行方式,即作为解释器参数,这时,第一行的解释器信息,失效
eg:`python test.py`
